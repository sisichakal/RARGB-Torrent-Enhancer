RARGB Torrent Enhancer (Userscript)

RARGB Torrent Enhancer is a Tampermonkey userscript that improves the RARGB torrent listing pages by adding inline thumbnails, hover image previews, and direct magnet download links.

Features

Adds a Thumbnail column to torrent listings

Displays up to 5 preview images directly in the table

Shows all available screenshots in a floating preview when hovering over a row

Adds a Magnet Download column with one-click access

Click on any thumbnail to start the magnet download

Automatic caching of torrent detail pages to reduce requests

Rate-limited requests to avoid overloading the site

Clean UI integration matching the original RARGB layout
